## Documentation Format

From version 4.20.58 onwards, all documentation follows:
- No code examples
- Strategy-based approach
- File path references
- Clear restrictions (❌ DO NOT, ❌ NEVER, ❌ AVOID)
- Mandatory rules (✅ ALWAYS, ✅ MUST, ✅ SHOULD)
- AI Agent Guidelines
