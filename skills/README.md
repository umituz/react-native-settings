# React Native Settings Skills

Claude Code skills for `@umituz/react-native-settings` - Complete settings hub.

## Installation

```bash
npx skills add /Users/umituz/Desktop/github/umituz/apps/mobile/npm-packages/react-native-settings/skills/SKILL.md -g
```

## Usage

Say: **"Setup settings screens in my app"**

## Features

- **Settings Stack** - Complete navigation system
- **Localization** - Multi-language support (i18next)
- **Appearance** - Theme switching (light/dark/system)
- **Notifications** - Preference management
- **Gamification** - Achievements and progress
- **Star Ratings** - App store prompts
- **FAQs** - Built-in FAQ screen
- **Feedback** - Feedback forms
- **Legal** - Privacy, terms screens

## Dependencies

- @react-navigation/native, @react-navigation/stack
- i18next, react-i18next
- expo-constants, expo-device, expo-haptics
- expo-localization, expo-notifications, expo-store-review
- @umituz/react-native-design-system
